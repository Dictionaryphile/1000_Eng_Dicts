部分词典我都会有一个专贴，整理相关信息。
- 查词典可以解决绝大部分语言方面的问题。
- 一本牛津高阶，顶一百个英语名师。
- 老师就是比你先查了词典而已。
- 知识越多的人，查词典越多。
- 不查词典，不学习的人，是可耻的。
- 其实不管学什么，最终都是把自己变成一本相关领域的活词典，仅此而已。
- 一切问题，都是语言和逻辑的问题。
